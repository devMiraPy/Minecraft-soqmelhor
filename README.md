🎮 Site inspirado em Minecraft

Um projeto front-end inspirado no visual oficial do Minecraft, desenvolvido com foco em design moderno, estética pixel-art e experiência imersiva. O site apresenta seções interativas, animações suaves e uma interface totalmente temática baseada no universo do jogo.
